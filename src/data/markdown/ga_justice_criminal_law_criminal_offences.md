#  Cionta Choiriúil

  * [ Coireanna a aicmiú i gcúiseanna coiriúla ](/ga/justice/criminal-law/criminal-offences/classification-of-crimes-in-criminal-cases/)

Déantar imscrúdú ar chion coiriúil agus déantar é a ionchúiseamh bunaithe ar
an gcineál coire atá i gceist. Chun na críche seo, féadtar cionta coiriúla a
aicmiú i slite difriúla.

  * [ Cionta oird phoiblí ](/ga/justice/criminal-law/criminal-offences/public-order-offenses-in-ireland/)

Tá an chuid is mó den dlí a bhaineann le cionta oird phoiblí leagtha amach san
Acht um Cheartas Coiriúil (Ord Poiblí), 1994. Faigh amach níos mó faoi na
cineálacha difriúla cionta.

  * [ Alcól agus an dlí ](/ga/justice/criminal-law/criminal-offences/alcohol-and-the-law/)

Pléann an cháipéis seo an dlí maidir le alcól a dhíol in áitribh chéadúnaithe
agus siopaí eischeadúnais, ól in áiteanna poiblí agus tiomáint faoi thionchar
alcóil.

  * [ An dlí maidir le gnéas a dhíol nó a cheannach ](/ga/justice/criminal-law/criminal-offences/the-law-on-buying-and-selling-sex/)

Faigh amach faoi do chearta nuair a cheannaíonn tú dearbhán bronntanais.

  * [ An dlí ar chaint fuath ](/ga/justice/criminal-law/criminal-offences/law-on-hate-speech/)

Leagann an tAcht um Thoirmeasc ar Ghríosú chun Fuatha 1989 amach an dlí maidir
le fuathchaint in Éirinn.

  * [ Tinte Ealaíne: an dlí ](/ga/justice/criminal-law/criminal-offences/the-law-on-fireworks/)

Ní ceadmhach ach na tinte ealaíne is lú baolaí a dhíol leis an bpobal. Tá
cumhachtaí ag an nGarda Síochána anois daoine a ghabháil má tá tinte ealaíne
ina seilbh acu gan cheadúnas. Tuilleadh eolais anseo.

  * [ Cionta drugaí ](/ga/justice/criminal-law/criminal-offences/drug-offences/)

In Éirinn is é an tAcht um Mí-Úsáid Drugaí 1977 arna leasú ag reachtaíocht
eile an phríomhreachtaíocht a chlúdaíonn cionta drugaí agus na pionóis faoi
seach.

  * [ Cionta Tíomána ](/ga/travel-and-recreation/motoring/driving-offences/driving-offences/)

Tá eolas anseo maidir leis na pionóis a ghearrtar má fhaightear ciontach tú i
gCoir Tiomána.
