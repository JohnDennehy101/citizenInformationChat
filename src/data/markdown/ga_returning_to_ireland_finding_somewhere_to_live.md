#  Áit éigin le cónaí ann a aimsiú

  * Overview 

[ Teacht ar áit éigin le maireachtáil ar fhilleadh duit go hÉirinn
](/ga/returning-to-ireland/finding-somewhere-to-live/finding-somewhere-to-
live-on-your-return-to-ireland/)

Soláthraíonn an doiciméad seo faisnéis faoi chóiríocht a aimsiú nuair a
fhilleann tú ar Éirinn.

  * [ Tacaíochtaí tithíochta d’eisimircigh Éireannacha atá ag filleadh ](/ga/returning-to-ireland/finding-somewhere-to-live/guide-to-housing-supports-for-returning-irish-emigrants/)

Más saoránach Éireannach thú atá ag filleadh abhaile ón iasacht, b’fhéidir go
mbeadh cabhair uait le lóistín. Féach tacaíochtaí le ligean ar cíos agus le
ceannach, chomh maith le tithíocht shóisialta agus lóistín éigeandála.

  *   * [ Cearta agus oibleagáidí na dtionóntaí ](/ga/housing/renting-a-home/tenants-rights-and-responsibilities/tenants-rights-and-obligations/)

Eascraionn príomhchearta agus príomhfhreagrachtaí tionóntaí atá i gcóiríocht
phríobháideach ar cíos ó dhlí tiarnaí talún agus tionóntaí, mar aon le haon
chomhaontú tionóntachta i scríbhinn nó ó bheil.

**This document is in:** [ Cearta agus oibleagáidí na dtionóntaí
](/ga/housing/renting-a-home/tenants-rights-and-responsibilities/)

  * [ Na cearta ag atá tiarnaí talún agus na hoibleagáidí atá orthu ](/ga/housing/renting-a-home/landlords-rights-and-responsibilities/landlords-rights-and-obligations/)

Tá cearta áirithe ag úinéirí talún agus tá oibleagáidí áirithe orthu a
dhíorthaíonn ón dlí um thiarnaí talún agus tionóntaí agus ó aon chomhaontú
tionóntachta freisin (bíodh sé scríofa nó labhartha) idir an tiarna talún agus
an tionónta.

**This document is in:** [ Cearta agus oibleagáidí tiarnaí talún
](/ga/housing/renting-a-home/landlords-rights-and-responsibilities/)
