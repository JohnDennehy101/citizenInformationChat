###  My Computer My Way

[ My Computer My Way ](https://mcmw.abilitynet.org.uk/) is an AbilityNet run
website that includes articles explaining how to use the accessibility
features built into your computer, tablet or smartphone. The site is regularly
updated as new features and changes are made to the Windows, MacOS, iOS,
Chrome OS and Android operating systems. The site is broken down into the
following sections:

  * [ Vision ](https://eur03.safelinks.protection.outlook.com/?url=https%3A%2F%2Fmcmw.abilitynet.org.uk%2Fimpairment%2Fvision&data=05%7C02%7CAlan.OConnor%40ciboard.ie%7Cfc3ee1818e7e4c5dd34008dc90544b5b%7Cbb4e61ce39a3414abb5d6a0290c95fd9%7C0%7C0%7C638543938916557473%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C0%7C%7C%7C&sdata=Nb%2FqvpzMQKkCvRjVSONZviUCt5bTstSi8e%2FlO6Vop0M%3D&reserved=0) – computer adjustments to do with vision and colour 
  * [ Hearing ](https://eur03.safelinks.protection.outlook.com/?url=https%3A%2F%2Fmcmw.abilitynet.org.uk%2Fimpairment%2Fhearing&data=05%7C02%7CAlan.OConnor%40ciboard.ie%7Cfc3ee1818e7e4c5dd34008dc90544b5b%7Cbb4e61ce39a3414abb5d6a0290c95fd9%7C0%7C0%7C638543938916571479%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C0%7C%7C%7C&sdata=CQtNw2jDf%2FpABFQdALM7EpmxXF7D%2B6R5UJW1LmheAGo%3D&reserved=0) – computer adjustments to do with hearing, communication and speech 
  * [ Motor ](https://eur03.safelinks.protection.outlook.com/?url=https%3A%2F%2Fmcmw.abilitynet.org.uk%2Fimpairment%2Fmotor&data=05%7C02%7CAlan.OConnor%40ciboard.ie%7Cfc3ee1818e7e4c5dd34008dc90544b5b%7Cbb4e61ce39a3414abb5d6a0290c95fd9%7C0%7C0%7C638543938916585202%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C0%7C%7C%7C&sdata=s3qNszeZXoXwI4reiuq9zXw%2B%2FYm8QzwpiZGa0L12uQQ%3D&reserved=0) – computer adjustments to do mobility, stamina and dexterity 
  * [ Cognitive ](https://eur03.safelinks.protection.outlook.com/?url=https%3A%2F%2Fmcmw.abilitynet.org.uk%2Fimpairment%2Fcognitive&data=05%7C02%7CAlan.OConnor%40ciboard.ie%7Cfc3ee1818e7e4c5dd34008dc90544b5b%7Cbb4e61ce39a3414abb5d6a0290c95fd9%7C0%7C0%7C638543938916596325%7CUnknown%7CTWFpbGZsb3d8eyJWIjoiMC4wLjAwMDAiLCJQIjoiV2luMzIiLCJBTiI6Ik1haWwiLCJXVCI6Mn0%3D%7C0%7C%7C%7C&sdata=MtolJtEpWPwasH1mYGpn2zUyIXxHKeZqb99a3t%2Ft294%3D&reserved=0) – computer adjustments to do with attention, learning and memory
