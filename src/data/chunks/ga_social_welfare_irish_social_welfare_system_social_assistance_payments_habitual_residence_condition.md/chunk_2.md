##  Réamhrá

Tá an córas leasa shóisialaigh in Éirinn roinnte i dtrí phríomhchinéal
íocaíochtaí. Is iad seo:

  * Íocaíochtaí árachais shóisialaigh (mar shampla, [ Sochar Cuardaitheora Poist ](/ga/social-welfare/unemployed-people/jobseekers-benefit/) ) 
  * Íocaíochtaí faoi thástáil mhaoine nó íocaíochtaí cúnamh shóisialta: (mar shampla, [ Liúntas Cuardaitheora Poist ](/ga/social-welfare/unemployed-people/jobseekers-allowance/) ) 
  * Íocaíochtaí uilíocha: (mar shampla, [ Sochar Linbh ](/ga/social-welfare/families-and-children/child-benefit/) ) 

Le gach íocaíocht leasa shóisialaigh in Éirinn, ní mó duit na rialacha maidir
le gach scéim a chomhlíonadh chun a bheith cáilithe. Caithfidh tú gnáthchónaí
in Éirinn a bheith agat chun cáiliú do na híocaíochtaí seo a leanas:

  * [ Pinsean na nDall ](/ga/social-welfare/families-and-children/back-to-work-family-dividend/)
  * [ Díbhinn Teaghlaigh um Fhilleadh ar Obair ](/ga/social-welfare/disability-and-illness/blind-persons-pension/)
  * [ Liúntas Cúramóra ](/ga/social-welfare/carers/carers-allowance/)
  * [ Sochar Linbh ](/ga/social-welfare/families-and-children/child-benefit/)
  * [ Liúntas Míchumais ](/ga/social-welfare/disability-and-illness/disability-allowance/)
  * [ Liúntas Cúraim Bhaile ](/ga/social-welfare/disability-and-illness/domiciliary-care-allowance/)
  * [ Íocaíocht Caomhnóra (Neamh-ranníocach) ](/ga/social-welfare/death-related-benefits/guardian-and-orphan-payments/)
  * [ Liúntas Cuardaitheora Poist ](/ga/social-welfare/unemployed-people/jobseekers-allowance/) agus Íocaíocht Idirthréimhseach [ liúntas Cuardaitheora Poist ](/ga/social-welfare/unemployed-people/jobseekers-transitional-payment/)
  * [ Íocaíocht Teaghlaigh Aon Tuismitheora ](/ga/social-welfare/families-and-children/one-parent-family-payment/)
  * [ Pinsean Stáit (Neamh-Ranníocach) ](/ga/social-welfare/older-and-retired-people/state-pension-non-contributory/)
