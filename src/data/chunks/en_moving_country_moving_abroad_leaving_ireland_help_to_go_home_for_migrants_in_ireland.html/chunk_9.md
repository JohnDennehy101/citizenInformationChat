##  How do I apply for the voluntary return scheme?

To apply for the voluntary return scheme, you can:

  * Download the [ Voluntary Return Application Form ](https://www.irishimmigration.ie/wp-content/uploads/2024/05/Voluntary-Return-Application-Form.pdf) or 
  * Email voluntaryreturns@justice.ie
