#  Taisteal thar Lear

  * [ Cárta Árachais Sláinte na hEorpa ](/ga/travel-and-recreation/travel-abroad/european-health-insurance-card/)

Tugann an Cárta Eorpach Árachas Sláinte rochtain ar leighis is gá, stát-ar
fáil cúram sláinte le linn fanacht go sealadach i dtír eile den AE / LEE.

  * [ Taisteal thar lear agus vacsainiú ](/ga/travel-and-recreation/travel-abroad/vaccinations-health-tips/)

Tá an vacsaín éigeantach chun dul isteach roinnt tíortha. Eolas ar vacsaíní do
dhaoine a chónaíonn in Éirinn má tá siad ag dul thar lear.

  * [ Teastas Digiteach COVID an AE (DCC) ](/ga/travel-and-recreation/travel-abroad/digital-covid-certificate/)

Forbhreathnú ar an Teastas Digiteach COVID an AE (DCC) agus taisteal
neamhriachtanach ón 19 Iúil 2021.

  * [ Ag taisteal thar lear le páistí ](/ga/travel-and-recreation/travel-abroad/travelling-abroad-with-children/)

Sa doiciméad seo tugtar comhairle do thuismitheoirí agus iad ag taisteal thar
lear le páistí.

  * [ Ag taisteal thar lear ](/ga/travel-and-recreation/travel-abroad/travelling-abroad-during-covid/)

Faisnéis faoi cé atá in ann taisteal thar lear le linn COVID-19, cad atá le
déanamh sula dtéann tú agus conas a bheith cinnte go leanann tú na rialacha
san áit a bhfuil tú ag dul.
