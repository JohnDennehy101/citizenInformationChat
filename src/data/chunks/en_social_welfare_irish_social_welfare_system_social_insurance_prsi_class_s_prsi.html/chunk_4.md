##  Social welfare payments covered by Class S PRSI

Class S contributions only cover you for a limited number of payments. In
general, you are not covered for any short-term payments, such as illness and
disability payments.

If you pay Class S contributions and you meet all the other conditions for the
payment, you may be entitled to:

  * [ Maternity Benefit ](/en/social-welfare/families-and-children/maternity-benefit/)
  * [ Adoptive Benefit ](/en/social-welfare/families-and-children/adoptive-benefit/)
  * [ Paternity Benefit ](/en/social-welfare/families-and-children/paternity-benefit/)
  * [ Parent's Benefit ](/en/social-welfare/families-and-children/parents-benefit/)
  * [ Widow's, Widower's or Surviving Civil Partner's (Contributory) Pension ](/en/social-welfare/death-related-benefits/widowed-persons-contrib-pension/)
  * [ State Pension (Contributory) ](/en/social-welfare/older-and-retired-people/state-pension-contributory/)
  * [ Treatment Benefit Scheme ](/en/social-welfare/disability-and-illness/treatment-benefit-scheme/)
  * [ Invalidity Pension ](/en/social-welfare/disability-and-illness/invalidity-pension/)
  * [ Jobseeker's Benefit (Self-Employed) ](/en/social-welfare/unemployed-people/jobseekers-benefit-self-employed/)
  * [ Guardian's Payment (Contributory) ](/en/social-welfare/death-related-benefits/guardian-and-orphan-payments/)
  * [ Benefit payment for 65 year olds ](/en/social-welfare/older-and-retired-people/payment-for-people-retired-at-65/)
  * [ Partial Capacity Benefit ](/en/social-welfare/disability-and-illness/partial-capacity-benefit/)
