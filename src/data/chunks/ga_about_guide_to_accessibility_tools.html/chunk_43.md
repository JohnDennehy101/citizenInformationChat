###  **Read Aloud: A Text to Speech Voice Reader**

**Description:** Read Aloud converts webpage text to audio, making it easier
for users with visual impairments or reading difficulties to access content.

  * **How to install** : 
    1. Go to the [ Chrome Web Store ](https://chromewebstore.google.com/) . 
    2. Search for "Read Aloud". 
    3. Click "Add to Chrome". 

  * **Basic Usage** : 
    * Click the Read Aloud icon in the toolbar. 
    * The extension will read the page content aloud, and you can control playback with the on-screen controls.
