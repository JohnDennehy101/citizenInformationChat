##  Payments that require the HRC

You must satisfy the habitual residence condition (HRC) to apply for the
following social welfare payments:

  * [ Back to Work Family Dividend ](https://www.citizensinformation.ie/en/social-welfare/social-welfare-payments/families-and-children/back-to-work-family-dividend/)
  * [ Blind Pension ](https://www.citizensinformation.ie/en/social-welfare/social-welfare-payments/disability-and-illness/blind-persons-pension/)
  * [ Carer’s Allowance ](https://www.citizensinformation.ie/en/social-welfare/social-welfare-payments/carers/carers-allowance/)
  * [ Child Benefit ](https://www.citizensinformation.ie/en/social-welfare/social-welfare-payments/families-and-children/child-benefit/)
  * [ Disability Allowance ](https://www.citizensinformation.ie/en/social-welfare/social-welfare-payments/disability-and-illness/disability-allowance/)
  * [ Domiciliary Care Allowance ](https://www.citizensinformation.ie/en/social-welfare/social-welfare-payments/disability-and-illness/domiciliary-care-allowance/)
  * [ Guardian’s Payment (Non-Contributory) ](https://www.citizensinformation.ie/en/social-welfare/social-welfare-payments/death-related-benefits/guardians-payments#l1f4da)
  * [ Jobseeker’s Allowance ](https://www.citizensinformation.ie/en/social-welfare/social-welfare-payments/unemployed-people/jobseekers-allowance/)
  * [ Jobseeker’s Transitional Payment ](https://www.citizensinformation.ie/en/social-welfare/social-welfare-payments/unemployed-people/jobseekers-transitional-payment/)
  * [ One-Parent Family Payment ](https://www.citizensinformation.ie/en/social-welfare/social-welfare-payments/families-and-children/one-parent-family-payment/)
  * [ State Pension (Non-Contributory) ](https://www.citizensinformation.ie/en/social-welfare/social-welfare-payments/older-and-retired-people/state-pension-non-contributory/)
  * [ Supplementary Welfare Allowance ](https://www.citizensinformation.ie/en/social-welfare/social-welfare-payments/supplementary-welfare-schemes/supplementary-welfare-allow/) (other than once-off exceptional and urgent needs payments) 
  * [ Widow’s, Widower’s or Surviving Civil Partner’s (Non-Contributory) Pension ](https://www.citizensinformation.ie/en/social-welfare/social-welfare-payments/death-related-benefits/widowed-persons-contrib-pension/)
