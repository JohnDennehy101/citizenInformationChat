##  How to apply for the Household Benefits Package

You can [ apply online for the Household Benefits Package
](https://services.mywelfare.ie/en/topics/pensions-and-older-people/household-
benefits/) using MyWelfare.ie, if you have a [ MyGovID account ](/en/social-
welfare/irish-social-welfare-system/claiming-a-social-welfare-
payment/applying-for-social-welfare-payments-and-services-online/) .

If you cannot apply online, you can fill in the [ Household Benefit Package
application form (pdf)
](https://assets.gov.ie/19988/9590902b20fa4e41badfdabf1bc3cca8.pdf) and return
