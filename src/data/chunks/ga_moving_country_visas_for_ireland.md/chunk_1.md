#  Víosaí agus cead dul isteach go hÉirinn

  * [ Víosaí do mhic léinn ](/ga/moving-country/visas-for-ireland/student-visas/)

Teastaíonn víosaí ó dhaoine as tíortha áirithe atá ag iarraidh staidéar a
dhéanamh in Éirinn. Faigh tuilleadh eolais.

  * [ Víosaí do thurasóirí ag teacht go hÉirinn ](/ga/moving-country/visas-for-ireland/visas-for-tourists-visiting-ireland/)

Teastaíonn víosaí ó thurasóirí as tíortha áirithe má tá siad ag iarraidh
teacht go hÉirinn. Faigh tuilleadh eolais.

  * [ Víosaí do dhaoine ag teacht go hÉirinn le haghaidh gnó ](/ga/moving-country/visas-for-ireland/visas-for-business-people-visiting-ireland/)

Bíonn víosaí ag teastáil ó dhaoine áirithe a bhíonn ag iarraidh teacht go
hÉirinn le haghaidh gnó. Faigh tuilleadh eolais.

  * [ Riachtanais víosa le teacht isteach in Éirinn ](/ga/moving-country/visas-for-ireland/visa-requirements-for-entering-ireland/)

Teastaíonn víosa iontrála Éireannach bailí ó shaoránaigh tíortha áirithe sula
dtagann siad go hÉirinn. Faigh amach an dteastaíonn víosa uait chun teacht
isteach in Éirinn.

  * [ Permission to enter Ireland ](/ga/moving-country/visas-for-ireland/permission-to-land-in-ireland/)

All non-EEA nationals need permission to enter Ireland, even if they are not
required to have a visa to enter Ireland.
