#  Córas na gCúirteanna

  * Overview 

[ Cúirteanna ](/ga/justice/courts-system/courts/)

Tá eolas anseo faoí na cúirteanna agus na hoifigí a thugann tacaíocht dóibh.

  * [ An Chúirt Leanaí ](/ga/justice/courts-system/children-court/)

Bíonn an Chúirt Leanaí ag déileáil le leanaí faoi 18 mbliana d'aois agus
tháinig sé i bhfeidhm i 2002. Leagann Acht Leanaí 2001 amach na rialacha
difriúla maidir leis an gCúirt Leanaí.

  * [ An Chúirt Dúiche ](/ga/justice/courts-system/district-court/)

Tá eolas anseo faoin gCúirt Dúichet.

  * [ An Chúirt Chuarda ](/ga/justice/courts-system/circuit-court/)

Déantar cur síos anseo ar an gCúirt Chuarda.

  * [ An Ardchúirt ](/ga/justice/courts-system/high-court/)

Tugtar eolas anseo faoín Ardchúirt

  * [ An Chúirt Tráchtála ](/ga/justice/courts-system/commercial-court/)

Soláthraíonn an Chúirt Tráchtála taifeach díospóide feidhmiúla i gcásanna
tráchtála.

  * [ An Chúirt Choiriúil Speisialta ](/ga/justice/courts-system/special-criminal-court/)

Déanamh suas agus dlínse an Chúirt Choiriúil Speisialta.

  * [ An Chúirt Achomhairc ](/ga/justice/courts-system/court-of-appeal/)

Is í príomhfheidhm na Cúirte Achomhairc éisteacht le hachomhairc ón Ard-Chúirt
agus ón gCúirt Chuarda.

  * [ Cúirt Uachtarach na hÉireann ](/ga/justice/courts-system/supreme-court/)

Déanann an Cáipéis seo cur síos ar an gCúirt Uachtarach.

  * [ Oifig na mBreithneoirI Costas Dlíthiú ](/ga/justice/courts-system/office-of-the-legal-costs-adjudicators/)

Cur síos ar fheidhmeanna oifig an Mháistir Fómhais.

  * [ An tÚdarás Rialála Seirbhísí Dlí ](/ga/justice/courts-system/legal-services-regulatory-authority/)

Faisnéis faoin Údarás Rialála Seirbhísí Dlí (LSRA).

  * [ Nós imeachta um éilimh bheaga ](/ga/justice/courts-system/small-claims-court/)

Déantar cur síos sa doiciméad seo ar an chaoi a n-oibríonn nós imeachta um
éilimh bheaga.

  * [ Nós imeachta Eorpach um éilimh bheaga ](/ga/justice/courts-system/european-small-claims-procedure/)

Cuireann an Nós Imeachta Eorpach um Éilimh Bheaga bealach saor agus éasca ar
fáil do dhuine chun éileamh trasteorann a dhéanamh gan gá le haturnae a
fhostú.
